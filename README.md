# Эта репа создана для подгрузки файлов для программы на Авто выполнение квестов DBD

# Функционал:
- Авто выполнение любого квеста DBD
- Авто получение награды
- Авто выбор следующего квеста

# Особые возможности:
- Выполнение квестов за пртивоположные роли, тоесть вы можете выполнить задание убийцы за ыжившего.
- Возможность выбрать приоритетный архив для прохождения
- (Скоро) Возможность установить ограничение для выполнения заданий по крови.

# This rep is created to download files for the Auto DBD Quest Completion program

# Functionality:
- Auto completion of any DBD quest
- Auto receipt of rewards
- Auto selection of the next quest

# Special features:
- Completion of quests for opposite roles, meaning you can complete the killer's quest as a survivor.
- Ability to select a priority archive for completion
- (Soon) Ability to set a blood limit for completing quests.
